# 美国斩杀线二创视频项目

## 项目目标
围绕“牢A（斯奎奇大王）/ 美国斩杀线 / 金牌讲师 / 王伟恒甜甜圈”这些互联网语境，创作一支带有黑色幽默、底层现实和反差感的二创短视频。

## 当前方向
- 主主题：美国底层人民的艰辛生活
- 核心概念：美国“斩杀线”
- 情绪结构：前半段像美国梦宣传片，后半段逐渐撕开现实
- 梗点穿插：
  - 金牌讲师："又要到饭了兄弟们"
  - 王伟恒："甜甜圈"
- 目标风格：牢A式冷静叙事 + 地狱笑话式反差 + 纪实感素材拼接
- 角色视觉设定：已补充 3 个角色立绘描述，详见 `CHARACTERS.md`

## 已确认的网络语境（工作摘要）
### 1. 牢A / 斯奎奇大王 / 美国斩杀线
- “牢A”即“斯奎奇大王”，相关公共讨论里核心是“美国斩杀线”概念。
- 该概念常被用来描述：当财务状况、住房、医疗、身份、信用等跌破某个临界值后，人会迅速从“还能撑住”滑入“系统性坠落”的状态。

### 2. 金牌讲师 / 又要到饭了兄弟们
- “金牌讲师”在中文互联网语境里，核心是以美国街头求生、要饭、领救助食物等内容出圈。
- 经典口头禅是："又要到饭了兄弟们"。
- 这个梗的传播点不只是搞笑，而是“明明很惨，却还用一种很乐观很江湖的方式把日子活成段子”的反差。

### 3. 王伟恒 / 甜甜圈
- “甜甜圈”更像一个“美国梦入口”的符号。
- 相关网络叙事常把它和“美国梦幻灭”联系在一起：一开始看到的是新鲜、甜、美式日常；后面逐渐面对的是现实压力、合规限制、底层生存和精神消耗。

## 当前已产出的脚本版本

# 短视频压缩稿（60-90秒）

## 标题建议
- 美国没有穷人？不，美国有斩杀线
- 从甜甜圈到要到饭，美国梦只差一条线
- 你以为美国是天堂，牢A说那是斩杀线没砍到你

## 成片口播稿

### 开头 0:00-0:08
你以为美国最可怕的是穷？
不是。
牢A说，美国最可怕的，是斩杀线。
一旦掉下去，人不是慢慢变惨，
是直接从“还能活”切到“快出局”。

### 中段 0:08-0:22
很多人刚去美国的时候，看到的是另一面。
高楼、商场、快餐、甜甜圈，
看什么都觉得新鲜，
像是终于进了发达世界。
但问题是，这些东西只是入口，不是答案。

### 中段 0:22-0:38
真正的美国底层，不是单纯没钱。
是你一旦没工作、没房子、没保险、没信用，
整个系统都会开始排斥你。
你坐一会儿，可能被赶。
你睡车里，可能出事。
你想喘口气，现实都不一定给你地方喘。

### 梗插入 0:38-0:50
所以金牌讲师那句
“又要到饭了兄弟们！”
你别只当搞笑。
在很多人那儿，这都不是节目效果了，
这是生存播报。
能稳定搞到一口饭，已经算有本事。

### 反差段 0:50-1:05
还有王伟恒那个“甜甜圈”梗。
刚开始看，美国是甜的。
吃的、逛的、看的，都是甜的。
可后面呢？
后面很多人咽下去的，
就不是甜甜圈了，
是账单、失业、流浪，还有体面一点点掉光。

### 结尾 1:05-1:18
所以美国不是没有底层。
美国是底层一旦出现，
就会被现实高速处理。
甜甜圈当然是甜的。
但斩杀线，不甜。
